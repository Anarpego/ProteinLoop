# Feature Spec: Submission Bundle

## Goal

Package the lablab upload artifacts into one deterministic zip bundle with checksums and a manifest.

## User Value

The team can move, archive, and verify the final upload packet without manually comparing individual files.

## Functional Requirements

1. The repo shall include a stdlib-only bundle generator.
2. The bundle shall include lablab copy, video script, slide source, PowerPoint deck, cover image, demo video, demo evidence, generated lablab form JSON, final readiness report, and license.
3. The bundle shall include a manifest with file sizes and SHA-256 checksums.
4. The bundle shall be generated by a Make target.
5. The submission artifact validator shall verify the bundle exists and contains required entries.
6. When `submission/gemma-evidence.json` exists, the bundle shall include it and the manifest shall checksum it.

## Acceptance Criteria

1. `make submission-bundle` creates `submission/proteinloop-lablab-upload.zip`.
2. `make submission-check` validates the zip entries and manifest.
3. README documents the bundle.
